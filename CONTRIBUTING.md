# Contribuciones

Gracias por apoyar Aegis Fusion. Este proyecto es publico y buscamos mejorar
la seguridad del endpoint con parches frecuentes.

## Como contribuir

1) Crea un fork y una rama con un nombre claro.
2) Describe el problema o mejora en el PR.
3) Incluye pasos de prueba o evidencia (logs, capturas, comandos).
4) Evita subir binarios o muestras sensibles al repo publico.

## Estilo y calidad

- Mantener ASCII cuando sea posible.
- Evitar dependencias nuevas sin justificar el motivo.
- Preferir cambios pequenos y revisables.
- Documentar cualquier cambio de comportamiento.

## Reportes de bugs

Incluye:
- Version y sistema operativo.
- Pasos para reproducir.
- Resultado esperado y actual.
- Logs relevantes.
